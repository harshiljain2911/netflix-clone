# Netflix Clone

A basic Netflix clone homepage built with React, Vite, and Tailwind CSS v4.

## Features

- Fully responsive layout
- Modern scrollable movie rows with hover animations
- Dynamic navigation bar effect on scroll
- Hero banner with gradient overlays
- Using Tailwind CSS v4 natively with Vite

## Prerequisites

- Node.js installed on your machine (v18+)

## How to Run the Project Local

1. Go to the project directory:
   ```bash
   cd netflix-clone
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the URL provided in the terminal (usually http://localhost:5173).

## Folder Structure

- `src/components/` - Reusable UI components like Navbar, Hero, and Row
- `src/pages/` - Future page views (Home, Login, etc.)
- `src/services/` - Future API calls and configuration (e.g., TMDB setup)
- `src/utils/` - Helper functions and utilities
